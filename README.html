# VibeCoding E‑commerce 웹페이지 — 기술 명세서 ## 개요 이 문서는 **VibeCoding Shop** e‑commerce 랜딩 페이지의 설계 및 구현 세부 사항을 정의한다. 목표는 입문자들에게 반응형 레이아웃, 인터랙티브 제품 카드, 테마 전환, 오프라인 친화적 자산 처리를 보여주는 최소한이면서도 기능적인 템플릿을 제공하는 것이다. ## 목표 - 구조, 스타일, 동작을 포함한 단일 **HTML 파일** 제공. - 4개의 제품 카드가 포함된 반응형 **Trending Products** 그리드. - **랜덤 제품 이미지** 포함 (오프라인 SVG 기반 테크 디바이스 일러스트). - 각 제품별 **옵션 선택자** 구현 (드롭다운, 세그먼트 버튼, 색상 스와치). - **라이트/다크 모드 전환** 지원 및 localStorage에 저장. - **프로모션 배너** 및 CTA 버튼 제공. - 외부 의존성 없이 모던하고 깔끔하며 모바일 친화적인 디자인 보장. ## 수용 기준 1. **레이아웃 & 반응형** - 데스크톱: 4열 그리드. - 태블릿: 2열 그리드. - 모바일: 1열 그리드. - 브랜드와 테마 토글이 포함된 스티키 헤더. - 저작권 및 메타 정보가 포함된 푸터. 2. **제품 카드 구성 요소** - 제품 이미지 영역 (랜덤 SVG 일러스트). - 제목, 가격 범위, 평점(별). - 옵션 선택 영역: - *Select* 드롭다운 (예: 사양). - *세그먼트 버튼* (예: 메모리 용량). - *컬러 스와치* (예: 제품 색상). - CTA: “Add To Cart” 버튼. 3. **프로모션 배너** - 임베디드 SVG 텍스트가 포함된 그라디언트 배경. - 제품 그리드 아래에 배치. - 콜투액션 버튼 포함 (예: “Browse Tech Products”). 4. **스타일링** - 색상, 그림자, 모서리 반경을 위한 CSS 변수 정의. - 다크/라이트 모드 테마 지원. - 둥근 모서리, 소프트 섀도우, 시스템 폰트. - 일관된 간격과 타이포그래피. 5. **동작** - 실행 시 임베디드 SVG 중 랜덤으로 제품 이미지 선택. - 인터랙티브 컨트롤 (버튼/스와치 선택 상태). - 테마 토글은 localStorage에 사용자 선호를 저장. - 푸터에 현재 연도를 동적으로 삽입. ## 기술 세부 사항 - **언어**: HTML5 + 임베디드 CSS 및 JavaScript. - **의존성**: 없음 (CDN, 외부 라이브러리 없음). - **이미지**: 오프라인 사용을 위한 인라인 SVG(Data URI). - **접근성**: 시맨틱 HTML 요소, 적절한 ARIA 라벨. - **브라우저 지원**: 최신 에버그린 브라우저. ## 파일 정보 - **파일 이름**: `index.html` - **대체 문서**: 이 파일(`document_kr.md`)은 기술 명세서 역할을 한다. --- **명세서 끝**
